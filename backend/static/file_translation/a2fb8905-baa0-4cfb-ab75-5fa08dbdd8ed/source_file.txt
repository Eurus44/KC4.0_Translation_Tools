ໂຄງການ ສະຖານີທ່າອາກາດສະຍານສາກົນ ກາມແຣງ ໄດ້ເລີ່ມລົງມືກໍ່ສ້າງແຕ່ເດືອນກັນຍາ 2016, ດ້ວຍສະຖາປັດຕະຍາກຳທີ່ເປັນເອກະລັກສະເພາະທີ່ເລີ່ມມາຈາກຮັງນົກແອນ, ເຊິ່ງເປັນຜະລິດຕະພັນທີ່ເປັນເອກະລັກສະເພາະຂອງແຂວງ ແຄ໋ງຮ່ວາ.
